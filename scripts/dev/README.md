# 🛠️ 개발용 스크립트

개발 중 자주 사용하는 스크립트들

## 📜 스크립트 목록

### start-server.bat / start-server.sh
개발 서버를 시작하는 스크립트

**Windows 사용법:**
```cmd
cd scripts/dev
start-server.bat
```

**Linux/Mac 사용법:**
```bash
cd scripts/dev
./start-server.sh
```

### add-sample-data.js
브라우저 콘솔에서 실행하여 샘플 데이터 추가

**사용법:**
1. 애플리케이션을 브라우저에서 열기
2. F12로 개발자 도구 열기
3. 스크립트 내용 복사하여 콘솔에서 실행
